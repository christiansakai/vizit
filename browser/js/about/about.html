The VIZIT Code Visualizer is a tool which takes the user’s JavaScript code and creates a visualization representing the step-by-step execution of the program.

This tool was built using PythonTutor, an open source project by Philip Guo that you can find here.

The VIZIT Runtime Analyzer is a tool that allows users to compare the efficiencies of two functions by generating graphs and tables which clearly indicate their relative performances.

This was built using Benchmark.js a benchmarking library that tests the performance of code.
